# Nuro Optimizer

Landing page and release artifacts for **Nuro Optimizer**, a free macOS optimizer from [Sentium Nuro Labs](https://getsentium.com).

- 🌐 Site: [optimizer.getsentium.com](https://optimizer.getsentium.com)
- ⬇️  Latest release: [github.com/DonteWong/nuro-optimizer-releases/releases/latest](https://github.com/DonteWong/nuro-optimizer-releases/releases/latest)

The macOS app is a notarized Developer ID build. Universal binary (Apple Silicon + Intel). Requires macOS 13 or later.
